How to use Reader

  • Read a page — on any article, click the Reader toolbar icon, then Read this page.
  • Library — saved articles sit on the left. Click one to open it, 📌 to pin it to the top, ✕ to delete, Clear all to wipe everything.
  • Search — the box above the library filters by title, synopsis, or URL. Each row shows its highlight (✎) and note (🗒) counts.
  • Settings — the ⚙ button up top changes the theme, the theme-change animation (Clean, Fractals, Bubbles, and more), reading font, text size, column width, library sort, and sidebar mode.
  • Sidebar modes — Pinned (always open), Hover (thin strip that expands), Reveal (hidden until you brush the left edge), or Hidden (a small handle on the left brings it back).
  • Library toggle — the ‹ in the sidebar header hides the library; the small handle on the left edge brings it back.
  • Export — the ⬇ button saves the current article, highlights, and notes as a Markdown file.
  • Status — each library item shows Unread (red) or Read (green) once you reach the end. Reopening a fresh tab automatically picks back up wherever you left off.
  • Filter — the ▾ button beside the search box opens a menu to filter the library by Unread or Completed; press Save to apply or Cancel to discard.
  • Progress bar — the white line up top shows how far through the article you are.
  • Highlight — select any text in the article, then pick a colour from the popup.
  • Notes — choose Note while highlighting, or click an existing highlight to add or edit a note. Highlights with notes show a ✎; hover one to read its note.
  • Original — the ↗ button in the bottom-left corner opens the source page (or shows where an imported file came from) in a new tab.
  • Keyboard shortcuts — press ? any time for the full list (j/k to scroll, n/p to switch articles, / to search, f for focus mode, r to read aloud, t for contents…).
